# Сервис 1С:Напарник

{: .important-title }
> Важный момент
> 
> Интеграция с 1С:Напарник в версии 2026.1 находится на этапе раннего тестирования. Между конструктором и сервисом 1С:Напарник отсутствует интеграция. Сервис внедрен, как эксперементальный функционал с прицелом на его развитие в будущих релизах.

Для подключения сервиса необходимо:
* быть зарегистрированным its.1c.ru
* получить ключ доступа на сайте [code.1c.ai](https://code.1c.ai/tokens/)
* включить доступность 1С:Напарник в настройках PrintWizard
* ввести ключ доступа
* на форме элемента справочника **Макеты** нажать на пиктограмму чата с 1С:Напарник

Важная информация:
* На текущий момент токен доступа используется общий для всех пользователей, поэтому переписка доступна всем
* На текущий момент 1С:Напарник ничего не знает о конструкторе или текущей конфигурации. Тем не менее, он может подсказывать по синтаксису 1С, давать рекомендации по рефакторингу кода. Также может составить не сложный запрос или написать несложный алгоритм.
* Далее функционал интеграции может быть существенно изменен.
