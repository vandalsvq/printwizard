# Внешний вид

Внешний вид редактора табличного документа очень похож на знакомым многим Microsoft Excel (и ему подобные программы). 

Над самим табличным полем расположена командная панель. Командная панель имеет 2-а этажа. Полный формат командной панели доступен в режиме редактирования. Рассмотрим ее подробнее:

**1-й уровень командной панели**

| Команда | Описание |
|--|--|
| [Иллюстрация] | Перейти / отключить режим редактирования макета |
| [Иллюстрация] | Подменю для выбора источника загрузки формы макета. **Из макета объекта** / **Из общего макета** - выбор макетов из состава конфигурации. **Из файла на диске** - выбор файла формата mxl, с диска пользователя. |
| [Иллюстрация] | Отменить / повторить последнее действие |
| [Иллюстрация] | Открыть / скрыть панель свойств области макета |
| [Иллюстрация] | Создать / удалить область макета |
| [Иллюстрация] | Изменение типа заполнения текста ячейки (текст, параметр, шаблон) |
| [Иллюстрация] | Вставка элемента в макет табличного документа |
| [Иллюстрация] | Команды изменения области макета табличного документа |
| [Иллюстрация] | Изменить параметры страницы |

**2-й уровень командной панели**

| Команда | Описание |
|--|--|
| [Иллюстрация] | Настроить / увеличить / уменьшить размер щрифта текущей области макета |
| [Иллюстрация] | Включить / отключить настройку шрифта полужирный / курсив / подчеркнутый для текущей области макета |
| [Иллюстрация] | Переключить горизонтальное выравнивание текста в текущей области макета |
| [Иллюстрация] | Переключить вертикальное выравнивание текста в текущей области макета |
| [Иллюстрация] | Добавить или удалить отступ перед текстом |
| [Иллюстрация] | Изменить цвет фона, текста, линии текущей области макета |
| [Иллюстрация] | Изменение способа размешения текста в ячейке. **Авто** - выводит текст до пересечения с текстом в другой ячейке. **Переносить** - переносит текст внутри ширины ячейки. **Обрезать** - обрезает текст на ширину ячейки. **Забивать** - заменяет текст на "###" если он не влазит в ширину ячейки. |
| [Иллюстрация] | Изменение линии границы текущей области макета |
| [Иллюстрация] | Подменю всех команд редактирования макета табличного документа |
