# Элементы интерфейса

### Вкладки

| Вкладка | Назначение |
| --- | --- |
| Описание | Наименование, код, ссылка на ОКУД, дата и документ утверждения |
| Макет | Редактор визуального макета: для .mxl — встроенный редактор таблицы; для .docx — загрузка из внешнего файла |
| Объекты метаданных | Список объектов метаданных (Справочники, Документы), для которых предназначен макет; имя команды печати для каждого объекта |
| Запросы | Запросы к данным информационной базы; auto-генерируются при добавлении объекта метаданных |
| Наборы | Наборы данных на основе запросов; могут содержать дополнительные поля (нумератор, конструктор поля, формат и т.д.) и BSL-алгоритм обработки |
| Соединения | Иерархическое дерево связей между наборами (аналог JOIN между наборами разных типов; задаёт шапку/строки или объединение разнородных наборов) |
| Области | Области из шаблона с таблицей параметров: для каждого параметра указывается источник (поле набора, выражение, формула и т.д.) |
| События | Обработчики событий цикла формирования (8 событий, см. ниже) |
| Функции | Пользовательские функции многократного использования (имя, параметры, BSL-тело) |

### Командная панель

| Команда | Действие |
| --- | --- |
| Тестирование | Запускает формирование печатной формы через pw_Исполнитель; открывает результат в виде табличного документа |
| Отладка | Переключает режим детального логирования pw_Исполнитель; при включённом режиме после тестирования вкладка «Журнал событий» показывает трейс |
| Заблокировать / Разблокировать | Блокирует или снимает блокировку макета; перед блокировкой выполняется комплексная проверка |
| Разработка → Передать в основной | Только для образа разработки; копирует все текущие изменения в основной (заблокированный) макет |
| Разработка → Закрыть образ | Только для образа разработки; передаёт изменения в основной макет и удаляет образ |

### Редактор табличного документа (.mxl)

Встроенный редактор с поддержкой форматирования (шрифт, выравнивание, границы, цвет, объединение ячеек). Области задаются стандартными средствами 1С (именованные области в табличном документе). Загрузка из Excel: скопировать выделенную область листа и вставить — форматирование сохраняется; области и параметры нужно разметить вручную.

### Редактор офисного документа (.docx)

Загрузка из файла, подготовленного в Word или совместимом редакторе по правилам разметки PrintWizard. Области и параметры определяются автоматически при загрузке по специальным меткам в документе.
