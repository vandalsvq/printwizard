# Условия видимости

[Иллюстрация]
     Пример настройки условий видимости

Кроме настройки доступа, для команды можно задать условия видимости в зависимости от значений реквизитов владельца команды. Если условия видимости заданы на этапе разработки их можно отключить, но изменять нельзя. При этом пользователь может самостоятельно добавить новые условия.
