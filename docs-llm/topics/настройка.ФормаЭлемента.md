# Форма элемента

Для создания новой печатной формы переходим к пункту меню "PrintWizard" и выбираем "Макеты (PrintWizard)". 

[Иллюстрация]
    
Меню "PrintWizard"

В форме списка создаем новый элемент и переходим на основную форму редактирования макета.

[Иллюстрация]
    
Основная форма макета

Основные элементы конструктора представлены 6 закладками на форме:

| Имя закладки | Назначение закладки |
|--|--|
| Описание (см. topic: описание.Реквизиты)   | Основная информация о макете, базовая настройка, объекты и команды печати |
| Запросы (см. topic: запросы.СписокЗапросов)    | Запросы основной информации и параметры запроса |
| Наборы (см. topic: наборы.СписокНаборов)     | Наборы данных и дополнительные поля для использования в макете |
| Соединения (см. topic: соединения.ВнешнийВид) | Соединения наборов данных |
| Макет (см. topic: макет.ВнешнийВид)      | Настройка макета печатной формы, областей и параметров |
| События (см. topic: макрос.ПередИнициализацией)    | Алгоритмы обработчиков событий макета конструктора |
| Функции (см. topic: функции.ВнешнийВид)    | Произвольные функции доступные из любого алгоритма макета |
| Журнал          | Журнал событий подготовки макета (доступен в режиме отладки) |

Подробная информация о каждой закладке представлена в соответствующей главе.

См. также: описание.Реквизиты, запросы.СписокЗапросов, наборы.СписокНаборов, соединения.ВнешнийВид, макет.ВнешнийВид, макрос.ПередИнициализацией, функции.ВнешнийВид
