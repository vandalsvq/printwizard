# Назначение

Форма решает полный цикл разработки печатной формы: от создания визуального макета до блокировки, после которой кнопка «Печать» целевого объекта метаданных становится доступна пользователям. Форма состоит из нескольких тематических вкладок: описание, макет, объекты метаданных, запросы, наборы, соединения, области, события, функции. Каждая вкладка отвечает за отдельный аспект печатной формы.
