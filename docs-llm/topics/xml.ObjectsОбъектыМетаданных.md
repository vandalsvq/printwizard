# Objects — объекты метаданных

Каждый элемент `Objects` описывает объект 1С, для которого создана печатная форма.

```xml
<Objects pw_type="Object"
         Number="1"
         Key="a1b2c3d4-..."
         Kind="Документы"
         Name="РеализацияТоваровУслуг"
         FullName="Документы.РеализацияТоваровУслуг"
         UseCommand="true">
  <Command Id="print_cmd_1" Order="1"
           Presentation="Счёт на оплату"
           CheckPosting="false"
           SkipPreview="false">
    <FormsList>ФормаДокумента</FormsList>
    <FunctionalOptions>ИспользоватьПечать</FunctionalOptions>
  </Command>
</Objects>
```

| Атрибут | Описание |
|---|---|
| `Kind` | Вид коллекции метаданных — см. перечисление (см. topic: xml.Описание) |
| `FullName` | Полное имя: `Документы.РеализацияТоваровУслуг` |
| `UseCommand` | Создавать команду в меню «Печать» |

### Objects / Command

| Атрибут / Элемент | Обязателен | Описание |
|---|---|---|
| `Id` | ✅ | Идентификатор команды |
| `Order` | ✅ | Порядок в меню «Печать» |
| `Presentation` | ✅ | Надпись команды |
| `CheckPosting` | ✅ | Проверять проведение перед печатью |
| `SkipPreview` | ✅ | Печатать без предварительного просмотра |
| `<FormsList>` | | Список форм объекта, на которых размещается команда (по одному имени на элемент) |
| `<FunctionalOptions>` | | Список функциональных опций, управляющих видимостью команды (по одному имени на элемент) |
